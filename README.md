# RespawnCooldown
受委託於匿名伺服器

### 版本: 1.12.2

### 前置插件: Vault

## 演示
[![插件演示](https://img.youtube.com/vi/VnkZMglY6Lk/0.jpg)](https://www.youtube.com/watch?v=VnkZMglY6Lk)

## 功能
  - 重生傳送間隔
  - 自定義物品打開略過界面
  - 透過花費金錢或權限略過等待
  - 自定義略過界面
  - 可自定義重生位置
